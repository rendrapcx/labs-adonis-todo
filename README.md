## how to run
- git clone
- `npm ci` (install from lock pack) / npm install
- rename `.env.example` (modif seperlunya)
- node ace generate:key (kalo gak ada key)
- mysql: create database lucid;
- Userdb dan password sesuaikan
- node ace migration:run
- node ace serve
